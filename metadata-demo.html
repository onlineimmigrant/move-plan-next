üîç How to View Page Source

To see the actual metadata in your browser:

  1. Go to http://localhost:3002
  2. Right-click anywhere on the page
  3. Select "View Page Source" (or press Ctrl+U / Cmd+U)
  4. Look for the <head> section
  5. You'll see all the <meta> and <title> tags

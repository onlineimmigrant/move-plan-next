- **After**: `w-10 h-10` with `ring-2 ring-teal-500 ring-offset-2` on selection
- **Changes**:
  - Slightly smaller for better fit
  - Ring effect instead of scale (more professional)

#### **Toggle Switches**

- Added focus ring states
